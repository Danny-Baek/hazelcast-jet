Create a Cluster within Seconds

It's easy to get started with Jet. The nodes automatically discover each other to form a cluster, both in a cloud environment and on your laptop. This is great for quick testing and simplifies deployment and maintenance.

Build Fault-Tolerant Data Pipelines that Scale

Process your data using a rich library of transforms such as windowing, joins and aggregations. Jet keeps processing data without loss even when a node fails, and as soon as you add another node, it starts sharing the computation load. First-class support for Apache Kafka, Hadoop, Hazelcast and many other data sources and sinks.

Massively Parallel with Predictable Latency

Jet's core execution engine was designed for high throughput with low system overhead and latency. It uses a fixed-size thread pool to run any number of parallel tasks. The engine is based on coroutines that implement suspendable computation, allowing many of them to run concurrently on a single thread.

Single Java Binary

Jet is a single 12 MB Java archive (JAR). It’s lightweight enough to run on small devices, you can embed it into your application as just another dependency or deploy it as a standalone cluster. First-class support for Kubernetes is included.
